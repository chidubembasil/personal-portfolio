About

I build interfaces and the systems behind them. Most of my recent work has been on admin panels and internal tools — React and TypeScript on the frontend, Tailwind CSS for styling, and Laravel or FastAPI handling the backend depending on the project.

I like owning a feature end-to-end: shaping the API contract, wiring up the database, and getting the UI to feel solid on every screen size. A good chunk of my time also goes into debugging — tracking down validation errors, authentication bugs, and the small data-shape mismatches that break things in production.
